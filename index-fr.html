A propos de moi

Ce portfolio va vous informer sur ce à quoi ressemblent mes travaux passés, mes compétences et mes passe-temps.

Le but de mon portfolio est d'expliquer de la manière la plus concise possible, qui je suis, ce que j'aime et ce que je fais.

Je m'appelle Manuel, je viens de France et je possède une Licence Profesionnelle dans le développement web ainsi que 3 ans d'expériences professionnelles dans la création de sites web : 1 année en tant que développeur web et 2 en tant qu'intégrateur web.

N'hésitez pas à consulter le code source de ce portfolio (surtout dans les fichiers CSS et JS) : j'ai essayé plusieurs choses différentes pour varier le code.

Mes compétances

HTML

Introduction : Le squelette des pages web : c'est ce qu'est HTML.

Mon expérience : Utilisation de ses sémantiques, de ces balises... et toutes les dernières fonctionnalités d' HTML (HTML5) pour construire une page web, ou une application. J'ai également créé des templates Emailing GMail, Outlook...

Ce que j'aime : le début et la première chose quand on crée une page web pour faire d'un projet une réalité. C'est ce que j'apprécie le plus à propos d'HTML.

Passez votre souris par-dessus pour en voir son contenu

CSS

Introduction : CSS est utilisé pour styler, customiser, moderniser et créer des pages web responsives.

Mon expérience : J'utilise sa dernière version (CSS3) comme ses @Media Queries pour créer du contenu spécifique basé sur la taille de l'écran de l'appareil. J'utilise également Flex ou ses équivalents CSS pour afficher du contenu complexe tout en restant responsive ou Grid pour afficher un tableau sans style particulier... Je connais tous les principaux contenus CSS. J'ai également utilisé le framework Bootstrap en entreprise.

Ce que j'aime : styler un site web et essayer de le designer davantage et être content du résultat final.

Passez votre souris par-dessus pour en voir son contenu

JQuery

Introduction : Utilisé pour créer des pages dynamiques.

Mon expérience : Via beaucoup de projets web ou d'expériences que j'ai eues, j'ai utilisé ses fonctions pour automatiquement créer ou éditer des cookies web. Créer des scripts ajoutant automatiquement du contenu HTML ou CSS. Ses fonctions d'animations... principalement dans des applications web en entreprises (mais également sur des pages web normales).

Ce que j'aime : faire, par exemple un formulaire automatique d'une application web pour le rendre plus dynamique. C'est ce que j'apprécie le plus.

PHP

Introduction : Utilisé pour créer des applications web sur des serveurs web.

Mon expérience : Je l'ai utilisé énormément quand j'étais Programmeur Web en entreprise. Au début j'utilisais le PHP procédural. Finalement j'utilisais le PHP Orientés Objets en MVC. N'hésitez pas à voir mon contenu public sur ce sujet sur mon compte GitLab. J'ai également utilisé le framework Smarty. J'ai également travaillé sur le CMS Prestashop et surchargé ses classes (en créant des add-ons), pour ajouter des menus en BackOffice du CMS et ainsi interagir avec de nouvelles données de la base de données.

Ce que j'aime : Le fait de créer des programmes sans avoir besoin d'utiliser des langages logiciels. Manipuler des données d'une application web que j'ai créé avec une base de données en SQL via des fonctions PHP.
