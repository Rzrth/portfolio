HTML

「はじめに」Webページの骨格：これが HTML です。

「私の体験談」セマンティクス、タグ…最新の HTML (HTML5) を使ってウェブページやアプリを構築しています。メールテンプレート（GMail, Outlook 等）も作成しました。

「気に入っている点」新しいウェブプロジェクトで最初に使うものだから。そして Web を “生かす” ものだから、私はそれが一番好きです。

CSS

「はじめに」スタイル、カスタマイズ、モダン化、レスポンシブなウェブページをつくるためのものです。

「私の体験談」最新バージョン（CSS3）を使い、@Media Queries を使ってデバイス幅に応じた CSS を設定しました。Flex や Grid を使って複雑なコンテンツのレスポンシブ対応もしています。Bootstrap フレームワークも仕事で使ったことがあります。

「気に入っている点」ウェブサイトをスタイリッシュにして、最終結果に満足できるデザインを追求できるところです。

jQuery

「はじめに」ウェブページを動的にするために使います。

「私の体験談」多くのウェブプロジェクトや経験を通じて、その関数を使いました。Cookie 操作、自動で HTML や CSS を追加・編集、アニメーション機能…主に仕事の Web アプリケーションで使いました（通常のウェブページでも使っています）。

「気に入っている点」例えば Web アプリケーション上で動的なフォームを作ることができること。それが私が最も好きなことです。

PHP

「はじめに」ウェブサーバー上で Web アプリケーションを作るために使います。

「私の体験談」Web プログラマーだった仕事でよく使いました。最初はオブジェクト指向ではなく使っていましたが、最終的には OOP や MVC デザインパターンを使いました。GitLab の公開 MVC & OOP 小規模コンテンツを見てもらっても構いません。Smarty フレームワークも使いました。さらに Prestashop CMS でそのクラスをオーバーロードしてアドオンを作り、バックオフィスメニューを追加してデータベースと連携させた経験もあります。

「気に入っている点」ソフトウェア言語を使わずにプログラムを作れるということ。Web プログラムで作ったものから SQL データベース経由でデータを操作できる点が好きです。
